## 축 1. 배우자성이 천간에 있는가

### 1-1. 천간 배우자성 (일반)

| 항목 | 내용 |
|------|------|
| **판별축** | 배우자 星(정관·편관 / 정재·편재)이 천간에 놓였는가 |
| **원문 근거** | 「천간은 모두가 바라보는 위치입니다. 그래서 주로 명예적 성공 남편이 됩니다.」(p1) / 「천간에 있으니 사회적인 명예도 있는 배우자입니다.」(p1) / 「천간은 주로 명예속성, 유명속성, 공공속성으로 활동하는 배우자」(p7) |
| **배우자 사회 구조 의미** | 사회적으로 드러나는 위치. 명예·유명·공공 영역에서 활동하는 배우자 구조. 겉으로 보이는 사회적 자리가 강함. |
| **고객용 표현** | 사회적으로 보이는 자리와 명예가 있는 배우자 구조입니다. 밖에서 인정받는 모습이 두드러집니다. |
| **주의점** | 천간만으로 배우자 전체를 단정하지 말 것. 지지·기립·12운성·연월일시와 반드시 병행. p2에서 관성이 천간·년간에 있으면 「직장형태」로 쓰이기도 함 — 배우자 vs 직장 구별 선행. |

### 1-2. 천간 정관·정재 (바른 배우자)

| 항목 | 내용 |
|------|------|
| **판별축** | 천간에 정관(여) / 정재(남) |
| **원문 근거** | 「정관이니 … 천간에 있으니 사회적인 명예도 있는 배우자입니다.」(p1) / 「여자는 정관, 남자는 정재는 올바른 배우자」(p3) |
| **배우자 사회 구조 의미** | 바른 배우자 星 + 공개적 위치 → 사회적 명예와 본인 만족도가 함께 드러나는 구조. |
| **고객용 표현** | 사회적으로도 인정받고, 본인 기준에서도 올바른 배우자로 느껴지는 구조입니다. |
| **주의점** | 「올바른 배우자」= 본인 만족 기준(p1). 타인 시선과 다를 수 있음. |

### 1-3. 천간 편관·편재

| 항목 | 내용 |
|------|------|
| **판별축** | 천간에 편관(여) / 편재(남) |
| **원문 근거** | 「편관이니 … 천간에 있으니 사회적인 명예도 있는 배우자입니다.」(p1) / 「편재나 편관이나 한 때 잘 할 때에는 숨넘어가게 잘하다가 어느 한순간에 확하고 사라지기도」(p5) |
| **배우자 사회 구조 의미** | 사회적 명예는 있으나 음양 불균형 → 대외 활동·경제·애정 중 하나가 들쭉날쭉. |
| **고객용 표현** | 밖에서는 잘나가 보이지만, 한쪽이 비면 갑자기 공허해지는 배우자 구조입니다. |
| **주의점** | 「애인 같은 남편」(p1) 등 관계 운영 서술과 혼동 금지. 여기서는 천간=공개적 사회 위치 축. |

## 축 2. 배우자성이 지지에 있는가

### 2-1. 지지 배우자성 (일반)

| 항목 | 내용 |
|------|------|
| **판별축** | 배우자 星이 지지(년·월·일·시 지)에 놓였는가 |
| **원문 근거** | 「지지는 내실이 있는 모양입니다. 그래서 주로 경제적 실력 남편이 됩니다.」(p1) / 「지지에 있으니 경제적인 실력을 갖춘 배우자입니다.」(p1) / 「지지는 실력속성, 실속속성, 사익속성 등으로 활동하는 배우자」(p7) |
| **배우자 사회 구조 의미** | 겉보다 실속·경제력·현실 기반이 강한 배우자. 사회 활동이 실력·사익 중심. |
| **고객용 표현** | 겉보다 실속과 경제 기반이 있는 배우자 구조입니다. |
| **주의점** | 지지=숨은 실력이지만, 연월일시 위치(축 9)에 따라 활동 무대 규모가 달라짐. |

### 2-2. 지지 정관·정재

| 항목 | 내용 |
|------|------|
| **판별축** | 지지에 정관(여) / 정재(남) |
| **원문 근거** | 「정관이니 … 월지에 있으니 … 지지에 있으니 경제적인 실력을 갖춘 배우자입니다.」(p1) |
| **배우자 사회 구조 의미** | 바른 배우자 星 + 실속 위치 → 경제적 실력을 갖춘 안정적 배우자. |
| **고객용 표현** | 실속 있는 경제 기반을 갖춘, 본인에게 올바른 배우자 구조입니다. |
| **주의점** | 월지=결혼 적령기(p1). 시기·나이대와 함께 봐야 함. |

### 2-3. 지지 편관·편재

| 항목 | 내용 |
|------|------|
| **판별축** | 지지에 편관 / 편재 |
| **원문 근거** | 「편관이니 배우자에 대한 만족도는 불만족」(p1, 지장간 乙 예시) / 「편재이면 불안정」(p5) |
| **배우자 사회 구조 의미** | 실속·경제 영역에서도 들쭉날쭉. 만족도 허결. |
| **고객용 표현** | 실속은 있어도 한쪽이 비는 불안정한 배우자 구조입니다. |
| **주의점** | 지장간 내 편관과 지지 직접 편관을 구분할 것. |

## 축 3. 배우자성이 지장간에 있는가

### 3-1. 지장간 배우자성 (일반)

| 항목 | 내용 |
|------|------|
| **판별축** | 배우자 星이 지장간에만 존재하는가 |
| **원문 근거** | 「지장간은 두드러진 모양은 아니지만 지지 안에 있으니 경제적 실력의 남편입니다. 그런데 두드러진 모양이 아닙니다.」(p1) / 「지장간은 없는 것으로 쳐도 됩니다. … 대체로 숨어 지내는 배우자로 처리하면 됩니다.」(p7) |
| **배우자 사회 구조 의미** | 사회·가정 모두에서 두드러지지 않음. 숨어 지내거나 간헐적으로 드러남. |
| **고객용 표현** | 겉으로는 잘 안 드러나고, 운이 올 때만 모습이 보이는 배우자 구조입니다. |
| **주의점** | p7 「없는 것으로 쳐도 됨」= 판별 우선순위 낮음. 운·대운에서 드러날 수 있음. |

### 3-2. 지장간 배우자 — 여기(餘氣) 강한 경우

| 항목 | 내용 |
|------|------|
| **판별축** | 지장간 배우자 + 해당 지지 여기(餘氣)가 강함 |
| **원문 근거** | 「월지 지장간 乙木이 배우자입니다. 辰는 전월 卯의 여기가 강하므로 지장간에 있지만 나름 경제적인 실력이 있는 배우자입니다.」(p1) |
| **배우자 사회 구조 의미** | 지장간이지만 여기가 살아 있어 실력·경제 기반이 어느 정도 작동. |
| **고객용 표현** | 겉으론 약해 보여도, 실속은 어느 정도 있는 배우자 구조입니다. |
| **주의점** | 「한번 거처간? 배우자」(p1) 가능성 — 인연의 확정성 약함. |

### 3-3. 지장간 배우자 — 미약한 경우

| 항목 | 내용 |
|------|------|
| **판별축** | 지장간 배우자 + 해당 지지에서 기운 미약 |
| **원문 근거** | 「未에서 乙은 미약한 상태이므로 경제적인 실력이 미미한 배우자가 됩니다.」(p1) |
| **배우자 사회 구조 의미** | 경제·실속 기반이 약함. 사회적 역량도 미미. |
| **고객용 표현** | 실속과 경제 기반이 약한, 희미하게 존재하는 배우자 구조입니다. |
| **주의점** | 편관·겁재 중첩 시 「한번 거쳐간 배우자」(p1) 가능. |

## 축 4. 천간과 지지가 기립했는가

### 4-1. 천간·지지 기립 (일반)

| 항목 | 내용 |
|------|------|
| **판별축** | 배우자 星이 천간·지지 모두에 기립(通根·透干) |
| **원문 근거** | 「천간과 지지 모두가 기립된 형태의 남편은 명예와 경제적 실력을 두루 갖춘 남편이 됩니다. 명실 공히 성공 남편이 되는 것입니다.」(p1) / 「천간과 지지가 기립되면 실력가 배우자가 됩니다.」(p7) |
| **배우자 사회 구조 의미** | 명예(천간) + 실력(지지) 동시 충족. 사회·경제 모두 갖춘 배우자. |
| **고객용 표현** | 겉으로 드러나는 명예와 실속 있는 경제력을 함께 갖춘 배우자 구조입니다. |
| **주의점** | 기립 ≠ 무조건 긍정. 12운성·신살·형충파해에 따라 역량 발현 방식이 달라짐. |

### 4-2. 기립 + 년주 위치

| 항목 | 내용 |
|------|------|
| **판별축** | 년주에 배우자 星이 지지에 뿌리 두고 기립 |
| **원문 근거** | 「년주에 정관이 辛이 지지에 뿌리를 두고 기립해 있으니 사회적 명예와 경제적 실력을 두루 갖춘 배우자입니다.」(p3) |
| **배우자 사회 구조 의미** | 국가·대사회 단위 무대 + 명예·실력 동시. 이른 인연. |
| **고객용 표현** | 큰 사회 무대에서 명예와 실력을 함께 보여주는, 이른 시기에 만날 수 있는 배우자 구조입니다. |
| **주의점** | 대운·연령대에 따라 년주 기운 약화 → 12운성 변화(p3 입묘·태지). |

## 축 5. 뿌리가 있는가

### 5-1. 지지에 뿌리 (通根)

| 항목 | 내용 |
|------|------|
| **판별축** | 천간 배우자 星이 지지에 뿌리를 둠 |
| **원문 근거** | 「지지에 뿌리를 두고 기립해 있으니 사회적 명예와 경제적 실력을 두루 갖춘 배우자」(p3) |
| **배우자 사회 구조 의미** | 공개적 명예(천간)가 실속(지지)에 받침됨. 역량이 공허하지 않음. |
| **고객용 표현** | 겉으로 보이는 모습에 실속이 받쳐 주는 배우자 구조입니다. |
| **주의점** | 뿌리 위치(년·월·일·시 지)에 따라 활동 무대·시기가 달라짐. 뿌리 없이 천간만 있으면 「명예적」이나 실속 약할 수 있음(p1 천간-only 서술과 대비). |

### 5-2. 뿌리 없음 / 기립 미약

| 항목 | 내용 |
|------|------|
| **판별축** | 천간만 있고 지지 뿌리 없음, 또는 지장간만 존재 |
| **원문 근거** | 「지장간은 … 두드러진 모양이 아닙니다.」(p1) / 「미약한 상태이므로 경제적인 실력이 미미」(p1) |
| **배우자 사회 구조 의미** | 겉·속 모두 약하거나, 겉(천간)만 있고 실속(지지)이 받치지 못함. |
| **고객용 표현** | 드러나는 모습과 실속 사이에 간극이 있거나, 전체적으로 힘이 약한 배우자 구조입니다. |
| **주의점** | 운에서 일시적으로 기립·발현 가능(p7 지장간). |

## 축 6. 12운성이 무엇인가

> 원문 총칙: 「배우자의 사회적인 역량은 연월일시의 지지 12운성을 따라 유추」(p3) / 「배우자와 나와의 관계는 일지의 12운성」(p3) / 「12운성이 각각의 육친들이 보이는 모습이나 역량이 된다」(p6)

### 6-1. 12운성 — 판별 원칙

| 항목 | 내용 |
|------|------|
| **판별축** | 배우자 星을 연·월·일·시 지지에 대입한 12운성 |
| **원문 근거** | 「정재 辛金을 년지의 12운성에 대입해서 배우자 운을 볼 수가 있소」(p6) / 「그 辛金을 일지에 대입하는데 12운성을 참조해 보면 배우자의 역량이 드러나게 되오」(p4) |
| **배우자 사회 구조 의미** | 연령대·활동 무대별 배우자의 사회적 역량·역할·활동 강도. |
| **고객용 표현** | 나이와 활동 환경에 따라 배우자의 사회적 모습이 달라지는 구조입니다. |
| **주의점** | 연월일시 지지 12운성 = 사회적 역량 / 일지 12운성 = 부부 관계. 혼동 금지. |

### 6-2. 입묘(入墓)

| 항목 | 내용 |
|------|------|
| **판별축** | 배우자 星 12운성 = 입묘 |
| **원문 근거** | 「입묘를 하게 됩니다. 이때에는 배우자의 사회적인 역량이 다소 위축이 됩니다. 배우자의 역할도 무대 뒤로 가야 합니다.」(p3) / 「일지 丑으로 들어오면 申金이 입묘를 합니다. … 가정에만 들어오면 … 꼼짝 못합니다.」(p7) |
| **배우자 사회 구조 의미** | 사회적 역량 위축. 무대 뒤·비공개. 가정 진입 시 더 위축. |
| **고객용 표현** | 밖에서는 역할이 줄고, 집에만 있으면 더 위축되는 배우자 구조입니다. |
| **주의점** | 주말부부·해외생활·거리 두기가 「완화」(p3, p4). 동거=병지 형태 따름(p4). |

### 6-3. 태지(胎地)

| 항목 | 내용 |
|------|------|
| **판별축** | 배우자 星 12운성 = 태지 |
| **원문 근거** | 「배우자의 모습이 寅에 태지가 됩니다. 가정에서 어린 애처럼 행동을 해야 하는 배우자가 됩니다.」(p3) |
| **배우자 사회 구조 의미** | 사회 활동 왕성·잘나가는 모습과 가정 내 역할(애처럼) 충돌. |
| **고객용 표현** | 밖에서는 활발하지만, 집에서는 어린아이처럼 행동해야 맞는 배우자 구조입니다. |
| **주의점** | 「그것이 배우자의 分」(p3) — 존재양식·역할 축. 사회·가정 이원 구조. |

### 6-4. 병지(病地)

| **고객용 표현** | 가까이 있을수록 건강·관계·역량이 위축되는 배우자 구조입니다. |

### 6-5. 목욕(沐浴)

### 6-7. 양지(養地)

## 축 9. 연월일시 어느 위치에 있는가

### 9-1. 년주

| 항목 | 내용 |
|------|------|
| **판별축** | 배우자 星·운성이 년주 |
| **원문 근거** | 「년주 … 젊은 날의 이성 … 국가단위의 큰 사회환경에서 활동하는 배우자」(p1) / 「년주에 있으면 이른 나이에 배우자 인연」(p3) |
| **배우자 사회 구조 의미** | 큰 사회 무대. 이른 인연. 1~20세 기운 강함(p3). |
| **고객용 표현** | 이른 시기에 만나고, 큰 사회 환경에서 활동하는 배우자 구조입니다. |

### 9-2. 월주

| 항목 | 내용 |
|------|------|
| **판별축** | 배우자 星·운성이 월주 |
| **원문 근거** | 「월주 … 시도단위의 사회환경에서 활동하는 배우자」(p1) / 「월지는 21~40세를 의미」(p4) / 「정재가 월지라서 사회활동을 한다면 내실이 있는 직업환경」(p7) |
| **배우자 사회 구조 의미** | 지역·시도 단위 활동. 결혼 적령기. 내실 있는 직업환경 가능. |
| **고객용 표현** | 결혼 적령기에 만나고, 지역 사회에서 실속 있는 활동을 하는 배우자 구조입니다. |
| **주의점** | p2: 년간 정관=직장, 월간 편관=배우자 가능 — 관성 다수 시 구별. |

### 9-3. 일주

| 항목 | 내용 |
|------|------|
| **판별축** | 배우자 星·운성이 일주(일지) |
| **원문 근거** | 「일주 … 가정활동을 열심히 하는 배우자」(p1) / 「일지가 배우자 宮」(p7) / 「제일 좋은 것은 … 일지가 정재/정관」(p3) |

| **고객용 표현** | 가정에서 역할이 안정되고, 배우자 만족·사회 역량이 가장 좋은 구조입니다. |
| **주의점** | 「단지 바깥 활동에서 … 만족도」(p3) — 일지 밖 위치와 대비. |

### 9-4. 시주

### 9-5. 일지 밖 위치 (년·월지)
