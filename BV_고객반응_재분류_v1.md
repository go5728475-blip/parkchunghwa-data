## IV. **신규** **·** **고객** **자발** **반복** **문장**

### **정의**

> **상담** **종료** **후** **고객이** **자기** **언어로** **다시** **말한** **문장**

| | |
|---|---|
| **시점** | **종료** **후** *(퇴장** *직전** *~** *24h** *내** *기록** *)* |
| **조건** | **상담사** **문장** **암송** **✗** **·** **자기** **말** **로** **재구성** **●** |
| **코드** | **R** *(Repeat** *·** *자발** *)* |

**예**

| **상담사** **§** | **자발** **반복** **(R)** |
|----------------|---------------------------|
| 밖에서는 역할이 있었는데 집에만 있으면… | 「**남편이** **회사** **에선** **그랬는데** **집에선** **왜** **이래** **」** |
| 공간·시간·역할을 나눠야… | 「**우리** **는** **붙어** **있으면** **안** **되는** **부부** **구나** **」** |

## V. **우선순위** *(판정** *용** *)*

```
1순위  R  고객 자발 반복 문장     (있으면 ●●)
2순위  A4 > A3 > A2 > A1        (채택 반응)
       ─────────────────
       기억문장 O  →  기억문장 > 만족도
       기억문장 X  →  만족도 > 기억문장
       ─────────────────
제외   X1 · X2 · X3             (판정 미사용)
```

| **조건** | **우선** |
|----------|----------|
| **R** **있음** | **R** **> ** **A** **> ** **기억문장** **> ** **만족도** |
| **R** **없음** **·** **기억문장** **O** | **기억문장** **> ** **만족도** **> ** **X** |
| **R** **없음** **·** **기억문장** **X** | **만족도** **> ** **기억문장** **> ** **X** |
| **A1~A4** | **R** **·** **기억문장** **다음** **·** **X** **와** **별도** **집계** |

**기억문장** **O/X**

| | **판정** |
|---|----------|
| **O** | **종료** **후** **1문장** **구조** **회상** **가능** *(암송** *≠** *·** *의미** *포함** *)* |
| **X** | **회상** **실패** **·** **현상** **만** **·** **형식** **반복** **만** |
