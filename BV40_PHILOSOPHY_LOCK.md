## 2-B. 분 선언 — **시스템** **전제** *(LOCK** *추가** *)*

> **분** **선언** **은** **검증** **대상** **이** **아니다.**  
> **분** **선언** **은** **시스템** **전제** **이다.**

### 전제

| | |
|---|---|
| **시작** **형식** | 「**당신의** **○○** **의** **分은** **○○** **입니다** **.** **」** |
| **역할** | **이후** **운** **·** **현상** **·** **판단** **·** **방법** **의** **고정** **점** |
| **검증** **✗** | **분** **선언** **자체** **를** **LIVE** **·** **고객** **기억** **·** **재진술** **로** **검증** **하지** **않는다** |

### 분 선언 없으면

**운** **·** **현상** **·** **판단** **·** **방법** **은** **모두** **공중에** **뜬다.**

```
[분 선언] ──→ [운] ──→ [현상] ──→ [판단] ──→ [방법]
     ↑
  없으면 → 이하 전부 부양 (인과 단절)
```

### 적용 **영역** — **전** **부** **필수**

| 영역 | §1 **시작** **예** |
|------|-------------------|
| **직업** | 당신의 **직업** **의** **分은** **…** |
| **재물** | 당신의 **재물** **의** **分은** **…** |
| **사업** | 당신의 **사업** **의** **分은** **…** |
| **관계** | 당신의 **관계** **의** **分은** **…** |
| **결혼** | 당신의 **결혼** **의** **分은** **…** |
| **연애** | 당신의 **연애** **의** **分은** **…** |
| **자녀** | 당신의 **자녀** **의** **分은** **…** |

**관계** **상담** **은** **§2-A** **에** **따라** **§1** **고객** **분** **→** **§2** **배우자** **분** **을** **둔다** **(§2-A** **유지** **)** **.**

### LOCK **고정**

- **분** **선언** **=** **전제** **·** **필수** **·** **출력** **시작** **점**
- **분** **선언** **=** **검증** **항목** **✗** **·** **가설** **✗** **·** **LIVE** **PASS/FAIL** **대상** **✗**
- **이** **조항** **은** **더** **이상** **검증** **하지** **않는다** **—** **전제** **로** **고정** **한다**
