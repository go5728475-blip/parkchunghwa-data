## 2. 해석

### 2-1. 기억 = 「원인 설명」

고객이 **§1 주분** **이** **아니라** **§2·§3** **인과** **를** **기억**함.

| 기억층 | 내용 | 고객 반응 |
|--------|------|-----------|
| §1 주분 | 공간·역할… 오래 가는 | **기억** **약** |
| §2·§3 | 자녀 이탈 · 은퇴 · 붙임 · 앙금 · 삼식이 | **「그래서** **그런** **거구나」** **=** **원인** **설명** |

→ **몰입** **은** **§3** **현상·인과** **에서** **발생** · **§1** **은** **레이블** **로** **남지** **않음**

### 2-2. 「어?」 N — 양날의 검

| 가능 | 의미 |
|------|------|
| **(+) 긍정** | **납득** **·** **저항** **없음** |
| **(−) 위험** | **§1** **을** **읽지** **않고** **넘어감** · **전문용어** **=** **배경** **잡음** |

**PARTIAL** **맥락** **에서는** **후자** **가능성** **↑**

### 2-3. 행동 P

| | |
|---|---|
| **의미** | **평화협정·각방·취미** **일부** **수용** · **전체** **실행** **또는** **「왜」** **연결** **약함** |
| **가설** | **행동** **=** **§5** **조언** **으로** **들림** · **주분** **과** **연결** **✗** |

### 2-4. 분 복기 — 되풀이 ≠ 이해

```
고객: 「공간과 역할이…」 (반복)
질문: 「그게 무슨 뜻이에요?」

```

## 3. 가설 (미확정)

> **「공간·역할」은 상담사 언어이고, 고객은 이를 「분」으로 인식하지 않는다.**

| | 상담사 프레임 | 고객 프레임 |
|---|---------------|-------------|
| **§1** | 배우자의 **分** = 공간·역할 | **「우리** **왜** **이러지」** **원인** **설명** |
| **희망** | *(주분과 정합)* | **「느슨한** **부부」** **·** **「각자** **생활」** |
| **기억** | 주분 1문장 | **§3** **스토리** |

**추가 가설**

1. **고객** **어휘** **=** **희망사항** **에** **이미** **있음** *(사생활 · 취미 · 각자 방)*  
2. **「공간·역할」** **=** **#12·#13·#14** **내부** **코드** **의** **한국어** **번역** **→** **고객** **말** **≠**  
3. **「분」** **질문** **자체가** **낯섦** · **되풀이** **=** **시험** **대응** **일** **수** **있음**
