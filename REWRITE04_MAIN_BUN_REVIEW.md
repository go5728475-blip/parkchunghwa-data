## 2. 문제 — 현상이 주분에 섞임

| 구분 | 내용 | §1에 있어야? |
|------|------|:------------:|
| **구조** | 밖 역할 vs 집 역할 · 대외·가정 **축** | ✓ |
| **조건** | 은퇴 · 집만 상주 | §2 운 |
| **현상** | 역할 줄음 · **식사·대기** · **삼식이** | **§3** |

**현재 §1에 들어 있는 것**

```
[과거] 밖 역할 있었는데
[조건] 집·은퇴 후
[구조] 역할이 줄고
[현상] 식사·대기만 남기 쉬운  ← §3 bleed
```

**결과**

- 고객 복기 시 **「식사·대기」** **·** **「삼식이」** **만** **남음** → **주분** **아닌** **증상** **기억**
- 「밖 역할 / 집 위축」 **구조** **축** **은** **흐림**
- 원문 #16 핵심 — **「밖=만족 · 집=등급 하락」** — **이** **아니라** **「밥만」** **으로** **기억**

**판정 (층 분리)**

| 테스트 | 결과 |
|--------|:----:|
| 1문장 형식 | PASS |
| 1문장 복기 **구조** 유지 | **PARTIAL · FAIL** |
| 주분 ≠ 현상 | **FAIL** |

| 안 | 예상 복기 | 1문장 | 구조 유지 |

| **현재** | 「…식사·대기만 남는…」「삼식이 되는…」 | △ | **FAIL** |

**현재 FAIL 패턴 (2개+ 또는 설명 필요)**

1. 「밖에선 일했는데」

→ **3조각** · **또는** **「삼식이」** **한** **단어** **로** **끝남**

**A · B PASS 패턴**

- A: 「**바깥 역할** **유지** **→** **생기**」 **1축**
- B: 「**사회적** **역할** **→** **안정**」 **1축**

## 8. 권고안 전체 흐름 (후보 B)

### 1. 분 선언

당신의 배우자는 **가정 안에만 머무르기보다 사회적 역할이 함께 있을 때 안정되는 배우자**입니다.

### 2. 현재 운

은퇴로 **사회적 역할이 끊기고** **집만** 남는 시기 · **24시간 붙은 동거** *(#06)*  
→ **이** **주분** **에** **불리** **한** **운**

### 3. 삼식이 현상

집만 남으면 **역할·역량이 줄고** **식사·대기** **만** **남기** **쉬움** *(#16 현상층)*  
→ **三食** **·** **서글픔** *(기존 §3 유지)*

### 4. 판단 · 5. 행동

**집 밖 2시간·역할 1개** = **주분** **운영** **(사회적** **역할** **복구**)
