# 유사형 구분 규칙 v1

> **용도:** **선언** **(§B)** **·** **§C** **조립** **시** **혼동** **방지**  
> **근거:** `원형20_3단연결표_전수판정_v1` **§III** **·** **BV-37** **9축**

## 0. 사용법

```
판별축 확정  →  §B 선언 1형 (또는 복합 시 순서 고정)  →  §C 조립
```

| 금지 | 허용 |
|------|------|
| **현상** **(삼식이·앙금** **)** **으로** **형** **대체** | **원형** **축** **키** **로** **형** **선택** |
| **유사** **§C** **만** **보고** **단정** | **BV-37** **히트** **축** **우선** |
| **한** **형** **에** **겹치는** **키워드** **만** **나열** | **아래** **「** **선언** **키** **」** **1줄** **로** **구분** |
